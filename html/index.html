HTML – Complete Beginner to Advanced Guide

Simple explanations • All important tags • Real examples • Outputs

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages.

  • HTML is not a programming language
  • It uses tags to define content
  • Browsers read HTML and display web pages

2. Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    Hello World
  </body>
</html>
Output: A web page showing Hello World

3. HTML Tags

HTML tags are written inside < >.

  • <tag> – opening tag
  • </tag> – closing tag

7. Images

<img src="image.jpg" alt="Sample Image" width="200">

Attributes:

  • src – image path
  • alt – alternate text
  • width / height – size
